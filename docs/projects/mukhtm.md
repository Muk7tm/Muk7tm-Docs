# MUKHTM

Minimalist portfolio website focused on black-and-white visual style, smooth animation, and interactive details.

## Quick Links

- [GitHub Profile](https://github.com/Muk7tm)

## Features

- Responsive layout for desktop and mobile
- Custom cursor and hover effects
- Animated grid background
- Scroll-triggered animations
- Project filtering and contact form support

## Technology Stack

- HTML5
- CSS3
- Vanilla JavaScript
- GSAP

## Local Run

1. Clone repository.
2. Open `index.html` directly or run a static server.

## Deployment

The source README indicates deployment is configured for Netlify.
