# Mk7WIN

Feature-packed Windows application built with Electron to combine app installs, gaming tweaks, optimization tools, monitoring, and AI-assisted troubleshooting.

## Quick Links

- [GitHub Repository](https://github.com/Muk7tm/Mk7WIN)
- [Latest Releases](https://github.com/Muk7tm/Mk7WIN/releases)
- [Security Policy](https://github.com/Muk7tm/Mk7WIN/blob/main/SECURITY.md)
- [License](https://github.com/Muk7tm/Mk7WIN/blob/main/LICENSE)

## Core Features

- Winget app installer with one-click installs and install-all workflow
- Gaming tweaks (Game Mode, Xbox Game Bar and cache-related optimizations)
- System tuning and cleanup tools
- PC specs and real-time CPU/RAM monitoring
- Built-in AI assistant for troubleshooting suggestions

## Installation

1. Download latest release from GitHub.
2. Run `Mk7WIN.Setup.1.2.0.exe`.
3. Launch the app and apply required tools/tweaks.

## Built With

- Electron.js
- Node.js
- JavaScript
- Tailwind CSS
- Windows Package Manager (winget)
- Python-based AI backend

## Security and Privacy

Mk7WIN is documented as local-first and does not collect personal data.
