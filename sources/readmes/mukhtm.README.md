# Portfolio Website

A minimalist black and white portfolio website with smooth animations and interactive elements.

## Features

- Responsive design that works on all devices
- Custom cursor with interactive hover effects
- Grid background with subtle animations
- Scroll-triggered animations
- Project filtering functionality
- Contact form

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- GSAP for animations

## Deployment

This site is configured for easy deployment on Netlify.

### Deploy to Netlify

[![Deploy to Netlify](https://www.netlify.com/img/deploy/button.svg)](https://app.netlify.com/start/deploy?repository=https://github.com/yourusername/portfolio)

## Local Development

To run this website locally:

1. Clone this repository
2. Open index.html in your browser

Alternatively, you can use a simple server:

```bash
# Install serve globally (if not already installed)
npm install -g serve

# Run the server
serve
```

The site will be available at `http://localhost:3000`
